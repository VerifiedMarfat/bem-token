*** DESCRIPTION ***

E-commerce site project


*** HOW TO RUN ***

Make sure you have the following grunt tasks installed in the machine:

sass
scsslint
jscpd
jshint
complexity
browserify
uglify
copy
watch
githooks


1) Open terminal (or equivalent), navigate to the project directory

2) Run two commands:
grunt dev - for all the necessary services
grunt githooks - for code validation before commiting

3) To run local php server, type command 'php -S localhost:XXXX' where XXXX is the port number.

4) You are all set up. Go do some coding!
